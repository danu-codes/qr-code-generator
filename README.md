# 📱 QR Code Generator (React)

A simple and modern QR Code Generator built with React.  
You can generate QR codes from any text or URL and download them instantly.

---

## 🚀 Features

- 🔤 Generate QR codes from text or URLs  
- 📏 Custom QR code size input  
- ⬇️ Download QR code as PNG  
- ⚡ Fast API-based generation  
- 🎨 Clean modern UI (light theme)

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- QR Code API (https://api.qrserver.com)
- CSS3
